# 错误码

| 错误码 | 中文描述                  |
| ------ | ------------------------- |
| 00000  | 正确                      |
| U0001  | 注册账号时用户名已经存在  |
| U0002  | 注册账号时 email 已经存在 |
| U0003  | 注册账号时手机号已存在    |
| U0004  | 注册账号时 email 格式错误 |

|        |                           |

|        |                           |
